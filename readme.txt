Easily calculate simple and compound interest with our free Interest Calculator. Estimate final balances, accrued interest, and growth schedules in seconds!

[https://www.calculator.io/interest-calculator/](https://www.calculator.io/interest-calculator/)

== Usage ==

Add the Interest Calculator shortcode to your page, post or sidebar:

`[calcio_interest_calculator]`

Add the following code to your template where you would like the Interest Calculator to appear:

`<?php display_calcio_interest_calculator(); ?>`

== Installation ==

1. Upload the Interest Calculator /calcio_interest_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Interest Calculator plugin through the "Plugins" menu in WordPress.
